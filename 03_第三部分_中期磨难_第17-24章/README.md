# 第三部分：中期磨难（第17-24章）

## 章节范围
第十七章至第二十四章，涵盖全书约20%的内容。

## 主要情节

第三部分进入取经的中期阶段，主要情节围绕五庄观事件、白骨精事件以及取经团队内部矛盾的激化展开。李长庚在这一阶段开始尝试将部分工作外包给其他神仙或妖怪，这种做法虽然提高了效率，却也带来了新的问题。五庄观事件是这一部分的亮点，镇元子这一人物的登场为故事增添了浓厚的喜剧色彩。

五庄观事件的起因是李长庚想要为观音造势，提升她在取经项目中的话语权。他找到好友镇元子商议，设计了一出"人参果树被推倒"的戏码，由观音扮演"救活果树"的关键角色。镇元子配合演出，不仅假装离开五庄观，还安排孙悟空去寻求各路神仙的帮助，最后由观音出面救活人参果树。这一事件被写成揭帖，广泛传播，大大提升了观音的声望。

然而，镇元子在这次合作中展现出了极强的"嘘呵之术"（自我包装能力），他趁机宣传自己的"地仙之祖"名号，还邀请了福寿禄三星、东华大帝、瀛洲九老等各路神仙前来"求情"，把一次简单的造势活动变成了自己的广告秀。李长庚虽然有些无奈，但也不得不承认镇元子的手段确实高明。

白骨精事件则展示了李长庚工作中的另一面。为了简化工作，他将白骨精这一劫难外包出去，采用最基础的"降妖除魔"方案。不料孙悟空因为通臂猿猴之死请假离开团队，导致劫难执行出现了问题。李长庚不得不临时找六耳猕猴来冒充孙悟空，完成"三打白骨精"的戏码。这一事件暴露了李长庚工作中的投机取巧倾向，也为后续情节的发展埋下了伏笔。

在这一阶段，取经团队内部的矛盾也逐渐激化。猪八戒与沙悟净之间存在宿怨，两人在宝象国事件中差点大打出手。奎木狼（黄袍怪）私自下凡与百花羞成婚，沙悟净见义勇为却反被擒，李长庚和观音介入后才将事件平息。这一事件最终以奎木狼被罚去兜率宫烧火了事，但其中涉及的恩怨纠葛却远没有结束。

## 关键事件时间线

**第十七章**：描写审查结束后，李长庚与观音关系的进一步改善，两人开始更加默契地合作。

**第十八章**：重点描写镇元子这一人物的登场以及五庄观事件的策划过程。

**第十九章**：详细叙述五庄观事件的执行，镇元子的嘘呵之术得到了充分的展现。

**第二十章**：讲述白骨精事件的安排，李长庚决定外包这一劫难。

**第二十一章**：描写孙悟空请假离开团队，李长庚得知通臂猿猴的死讯。

**第二十二章**：叙述李长庚追到花果山与孙悟空的对话，两人的关系有所缓和。

**第二十三章**：描述李长庚找太上老君借调金银童子，为后续的平顶山劫难做准备。

**第二十四章**：宝象国事件的开端，奎木狼与百花羞的关系被揭开，沙悟净被擒。

## 本部分特点

第三部分的叙事风格出现了明显的变化，五庄观事件中镇元子的喜剧色彩与后续情节的沉重形成了鲜明对比。李长庚在这一阶段展现出了他的精明和世故，但也暴露了工作中的投机取巧倾向。孙悟空这一人物的形象在这一阶段逐渐丰满起来，他不再是简单的"齐天大圣"，而是一个有着深刻内心创伤、对天庭充满不公的愤懑的角色。
